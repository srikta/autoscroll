# 🎸 AutoScroll – Chords & Lyrics Scroller

A simple, distraction-free web app that automatically scrolls chords and lyrics — perfect for musicians during practice or live performance.

---

## 🚀 Features

* 🎵 Paste your **chords + lyrics**
* ▶️ **Auto-scroll** with adjustable speed
* ⏱️ **Start delay** before scrolling begins
* 🔍 **Zoom in/out** for better readability
* 🌙 **Dark / Light mode**
* ✏️ Edit anytime without reloading
* 📱 Responsive design (works on mobile)

## ⚙️ How to Use

1. Paste your chords and lyrics into the input box
2. Click **Load**
3. Adjust:

   * Speed
   * Delay
   * Zoom
4. Click **Start** 🎶
5. Use **Pause / Reset / Edit** as needed

---

## 🌐 Deploy on GitHub Pages

1. Go to your repository
2. Click **Settings → Pages**
3. Select:

   * Source: `main`
   * Folder: `/root`
4. Save

Your site will be live at:

```
https://your-username.github.io/repo-name/
```
